# @spark-connect-js/connect

Generated TypeScript types from the [Spark Connect](https://spark.apache.org/docs/latest/spark-connect-overview.html) protobuf definitions.

Part of [spark-connect-js](https://github.com/prustic/spark-connect-js). Most users should install `@spark-connect-js/node` instead — this package is a low-level dependency.

## Install

```bash
npm install @spark-connect-js/connect
```
