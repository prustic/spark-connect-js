# @spark-connect-js/connect

Generated TypeScript types and service definitions from the [Spark Connect](https://spark.apache.org/docs/latest/spark-connect-overview.html) protobuf spec.

> [!NOTE]
> This project is in early development. APIs may change between releases.

```bash
npm install @spark-connect-js/connect
```

```typescript
import { Plan, Relation, Expression, DataType } from "@spark-connect-js/connect";
import { SparkConnectService } from "@spark-connect-js/connect";
```

Used by [`@spark-connect-js/core`](https://www.npmjs.com/package/@spark-connect-js/core) and [`@spark-connect-js/node`](https://www.npmjs.com/package/@spark-connect-js/node). Most users don't need to install this directly.
